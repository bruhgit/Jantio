# Jantio - Java GUI Builder

Jantio, Java Swing ile geliştirilmiş sürükle-bırak özellikli bir GUI (Grafiksel Kullanıcı Arayüzü) oluşturucu uygulamadır.

## Özellikler

- **Sürükle-Bırak Tasarım**: Bileşenleri paletten tasarım alanına sürükleyip bırakarak kolayca arayüz oluşturabilirsiniz.
- **Bileşen Paleti**: JButton, JLabel, JTextField, JTextArea, JCheckBox, JRadioButton, JComboBox ve JPanel gibi yaygın Swing bileşenlerini içerir.
- **Özellik Paneli**: Seçili bileşenin konumunu (X, Y) ve boyutunu (Genişlik, Yükseklik) düzenleyebilirsiniz.
- **Kod Üretimi**: Tasarladığınız GUI'yi Java koduna dönüştürür ve dosyaya kaydedebilirsiniz.
- **Bileşen Yönetimi**: Eklenen bileşenleri liste üzerinden seçebilir, silebilir veya özelliklerini düzenleyebilirsiniz.

## Gereksinimler

- Java Development Kit (JDK) 17 veya üzeri
- Grafiksel arayüz için X11 display (Linux/Mac) veya uygun grafik ortamı

## Derleme

```bash
cd /workspace/Jantio
javac -d out src/jantio/*.java
```

## Çalıştırma

```bash
cd /workspace/Jantio
java -cp out jantio.Jantio
```

## Kullanım

1. **Bileşen Ekleme**: Sol taraftaki paletten bir bileşeni seçin ve tasarım alanına sürükleyip bırakın.
2. **Bileşen Taşıma**: Tasarım alanındaki bileşenleri fare ile sürükleyerek konumlarını değiştirebilirsiniz.
3. **Özellik Düzenleme**: Sağ taraftaki listeden bir bileşen seçerek özellikler panelinden konum ve boyut ayarlarını yapabilirsiniz.
4. **Kod Üretme**: "Generate Code" butonuna tıklayarak tasarladığınız GUI'nin Java kodunu oluşturabilirsiniz.
5. **Kodu Kaydetme**: Oluşturulan kodu "Save to File" butonu ile .java dosyası olarak kaydedebilirsiniz.
6. **Temizleme**: "Clear All" butonu ile tüm bileşenleri temizleyebilirsiniz.

## Üretilen Kod Örneği

Jantio ile tasarladığınız bir GUI şu şekilde Java koduna dönüştürülür:

```java
import javax.swing.*;
import java.awt.*;

public class GeneratedGUI extends JFrame {

    public GeneratedGUI() {
        setTitle("Generated by Jantio");
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setSize(800, 600);
        setLocationRelativeTo(null);
        setLayout(null);

        JButton button1 = new JButton("Button");
        button1.setBounds(50, 50, 100, 30);
        add(button1);

        JLabel label1 = new JLabel("Label");
        label1.setBounds(50, 100, 100, 30);
        add(label1);
    }

    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> {
            new GeneratedGUI().setVisible(true);
        });
    }
}
```
